# Spoken

A pocket reader that pulls clean text from any page. Built with Expo, Expo Router, and NativeWind.

## Quick start

```bash
pnpm install
pnpm start
```

- Text extraction uses the page DOM (`article`, `main`, or `body`).
- Try public pages without paywalls for best results.
